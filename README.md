# 🧠 ResumeMind

> 上传简历 PDF，AI 自动生成候选人评估报告 · 适合招聘初筛

ResumeMind 是一个轻量级的 AI 简历分析工具，帮助 HR 和面试官快速了解候选人核心画像、优势、风险及面试方向。它不抽取字段，而是直接输出一段有判断力的评估报告。


## ✨ 功能特点

- 📄 支持 PDF 简历上传（点击或拖拽）
- 🧠 AI 自动生成 5 模块评估报告：
  - 候选人画像
  - 核心优势（3-5 条）
  - 风险提示（2-3 条）
  - 面试建议（3-5 个问题）
  - 综合评估
- 📋 一键复制完整报告，便于存档或分享
- 🚀 纯前端实现，无需安装任何依赖


## 🖥️ 运行效果

1. 打开页面，上传 PDF 简历
2. 点击「开始分析」
3. 等待 3-5 秒，生成评估报告
4. 点击「复制报告」即可使用
